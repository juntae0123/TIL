# 사용자 정보 수정 페이지

## 프로젝트 소개
HTML과 CSS를 이용하여 사용자 정보를 수정하고 관리할 수 있는 웹 페이지를 구현한 프로젝트입니다.  
사용자는 프로필 사진, 자기소개, 투자 성향, 회원 정보, 금융 포트폴리오 등을 확인하고 수정할 수 있습니다.

---

## 주요 기능

### 1. 기본 정보 수정 (F301)
사용자의 기본 정보와 투자 성향을 설정할 수 있습니다.

- 프로필 사진 원형 출력
- 자기소개 입력 (textarea + 버튼)
- 투자상품 선호도 선택 (checkbox)
- 투자 스타일 선택 (radio)
- 관심 종목 선택 (checkbox)

---

### 2. 회원 정보 출력 (F302)
회원의 기본 정보를 입력하거나 확인할 수 있습니다.

| 항목 | 입력 방식 |
|-----|-----|
| 회원 번호 | readonly |
| ID | readonly |
| Email | email |
| Nickname | text |
| 나이 | number |
| 자산 및 연봉 | number |

---

### 3. 네비게이션 바 구현 (F303)
상단 네비게이션을 통해 페이지 이동이 가능합니다.

- 기본 정보 수정 페이지
- 포트폴리오 수정 페이지

---

### 4. 포트폴리오 수정 (F304)
회원의 금융 포트폴리오 정보를 입력 및 수정할 수 있습니다.

| 항목 | 입력 방식 |
|-----|-----|
| 회원 번호 | readonly |
| ID | readonly |
| 저축 성향 | radio |
| 최애 은행 | text |

---

### 5. 반응형 디자인 (F305)
화면 크기에 따라 레이아웃이 변경되도록 구현했습니다.

- **768px 이상**  
  - 프로필 사진과 투자 성향이 좌우 배치

- **768px 미만**  
  - 투자 성향 영역이 프로필 아래로 이동

---
