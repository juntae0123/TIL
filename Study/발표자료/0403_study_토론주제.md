# AI 모델 최적화 및 실무 활용 스터디 토론 자료

## 사전 학습 가이드: 반드시 알아야 할 핵심 개념

본격적인 토론에 앞서, 아래 나열된 개념들을 사전에 숙지해주세요! 각 파트별 키워드를 기술 블로그나 ai를 통해 미리 파악하고 오시기 바랍니다.

### 1. 모델 양자화 (Quantization) 관련 개념
* **부동소수점 vs 정수형 연산:** AI 가속기(NPU) 환경에서 FP32와 INT8 연산이 메모리 대역폭과 추론 속도에 미치는 차이를 이해해야 하기
* **양자화 매핑 파라미터:** 실수를 정수 공간으로 변환할 때 배율(Scale)과 영점(Zero-point)이 왜 수식적으로 필수적인지 파악하기
* **비트 쉬프팅 (Bit Shifting):** 고비용의 곱셈/나눗셈 연산을 비트 이동 연산으로 대체할 때 얻는 프로세서 레벨의 이점을 확인하기

### 2. 테스트 타임 도메인 적응 (TTA) 관련 개념
* **분포 이동 (Distribution Shift):** 훈련 데이터와 실전 테스트 데이터 간의 통계적 특성이 어긋나는 현상
* **배치 정규화 (Batch Normalization):** 입력 분포를 정규화하는 계층으로, TTA 기법 적용 시 주로 이 계층의 파라미터가 타겟이 되는 이유를 이해하기
* **엔트로피 최소화 (Entropy Minimization):** 모델의 예측 불확실성을 나타내는 엔트로피를 손실 함수로 사용하여, 테스트 단계에서 확신도를 높이는 학습 원리를 숙지하기

### 3. 적응적 센싱 (Adaptive Sensing) 관련 개념
* **OOD (Out-of-Distribution):** 모델이 학습하지 못한 도메인 밖의 데이터가 입력되었을 때 발생하는 성능 저하 문제
* **카메라 센서 파라미터의 영향:** 노출 시간, ISO, 조리개 값 등의 물리적 센서 제어가 이미지의 신호 대 잡음비(SNR)에 미치는 영향을 파악해보기

### 4. 도메인 지식 주입 및 의료 AI 관련 개념
* **귀납적 편향 (Inductive Bias):** 알고리즘이 보지 못한 데이터를 예측할 때 사용하는 가정입니다. CNN의 공간적 지역성(Spatial locality)과 연관 지어 이해해보기
* **비전 트랜스포머 (ViT):** 이미지를 패치 단위로 분해하고 Self-Attention을 수행하는 아키텍처
* **시계열 데이터의 2차원 변환:** 1D 생체 신호를 시각적 패턴(이미지)으로 변환하여 비전 모델에 태우는 방식의 공학적 이점을 고민해 보기

## 스터디 토론 주제 4선

### 토론 1. [양자화] "실수를 정수로 바꿀 때, 배율과 영점 조절은 왜 필수적인가?"
* **목표:** 양자화 공식의 수학적 근거와 하드웨어 최적화의 상관관계 파악.
* **토론 질문:**
    1. "단순히 소수점을 버리거나 반올림하는 방식을 쓰지 않고, 별도의 오프셋(Zero-point) 파라미터를 유지하는 이유는 무엇입니까?"
    2. "스케일링 파라미터를 비트 쉬프팅으로 근사하여 처리할 경우, 정밀도(Accuracy) 손실을 수반하게 됩니다. 실무자 입장에서 이 트레이드오프를 어떻게 평가하겠습니까?"

### 토론 2. [TTA] "가중치를 고정한 상태에서 모델은 어떻게 낯선 환경에 적응하는가?"
* **목표:** 분포 이동 문제 해결을 위한 동적 적응 기법의 장단점 비판.
* **토론 질문:**
    1. "조도 변화나 노이즈 발생 등 외부 환경 변화가 모델 내부 계층의 연산 결괏값에 구체적으로 어떤 통계적 왜곡을 초래합니까?"
    2. "일부 TTA 기법은 예측 확신도가 높은(엔트로피가 낮은) 샘플만 선택해 모델을 업데이트합니다. 이처럼 '쉬운 데이터'만 편식하여 학습하는 전략이 모델의 장기적인 범용성에 어떤 리스크를 가져올 수 있습니까?"

### 토론 3. [적응적 센싱] "모델을 키울 것인가, 입력 데이터를 최적화할 것인가?"
* **목표:** 소프트웨어 중심의 스케일링 법칙과 하드웨어 제어 기반 최적화의 효율성 비교.
* **토론 질문:**
    1. "사람의 육안에 선명하고 노출이 잘 된 이미지가 비전 AI 모델의 추론 정확도에도 항상 최적의 데이터라고 단언할 수 있습니까?"
    2. "경량화된 비전 모델에 적응적 센싱 기법을 결합했을 때, 단순히 파라미터를 수십 배 키운 초거대 모델보다 더 높은 성능 향상을 보였습니다. 이러한 결과가 시사하는 공학적 의미는 무엇입니까?"

### 토론 4. [의료 AI] "시계열 데이터를 굳이 이미지로 치환하여 분석하는 이유는 무엇인가?"
* **목표:** 도메인 전문가(의료진)의 작업 방식을 AI 아키텍처 설계에 반영하는 방법론 고찰.
* **토론 질문:**
    1. "수면 기사가 생체 파형의 '모양'을 시각적으로 인지하여 판독한다는 점이, 1D 시계열 분석 모델 대신 비전 트랜스포머(ViT)를 채택하는 데 어떤 기술적 타당성을 제공했습니까?"
    2. "에폭(30초 단위) 단위로 분할된 진단 결과를 다시 트랜스포머 인코더에 넣어 상호 어텐션(Cross-Attention)을 수행합니다. 시계열 데이터의 연속성을 복원하기 위한 이 후처리 과정의 역할은 무엇입니까?"
